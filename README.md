# Employee Task Manager (React + MUI)

Features:
- Mock authentication (Employee / Admin)
- CRUD tasks (create, edit, delete)
- Status updates (Pending → In Progress → Completed)
- Filtering, sorting, search by title
- Dashboard with progress bars and visual counts
- Dark / Light mode toggle
- Toast notifications using react-toastify
- Routing with React Router
- State management using Context API
- Forms with react-hook-form
- Uses localStorage as mock backend (no server)
- Sample unit test (Jest + React Testing Library)

How to run:
1. `npm install`
2. `npm start` (or `yarn start`)

Notes:
This is a scaffolded project created for the user. Modify/mock data in `src/utils/mockData.js` or use localStorage.
